About

This is a simple pokedex I built using PokeApi for my WDD 131 Course on dynamic web fundamentals. It's not supposed to be great, but it should work; my grade depends on it.

How To Use

Click on the "search" link in the navigation menu to go to the search page. When there, you can search for pokemon by name, type, or number. Results will load below the search bar. Click on a result to see more information, including a random pokedex entry for that pokemon!
